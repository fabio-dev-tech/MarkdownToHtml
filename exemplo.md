## 📦 Tabela com Recursos

| Recurso         | Suporte | Observações                          |
|-----------------|---------|--------------------------------------|
| **Títulos**     | ✅      | Usar `#`, `##`, ... até `######`     |
| **Listas**      | ✅      | Ordenadas `1.` ou com `-`            |
| **Códigos**     | ✅      | Inline com `` ` `` ou blocos com ``` |
| **Imagens**     | ✅      | `![alt](url)`                        |
| **Links**       | ✅      | `[texto](url)`                       |
| **Tabelas**     | ✅      | Suporte limitado, mas útil           |
| **Checkboxes**  | ✅      | Com `- [ ]` e `- [x]`                 |
| **Emojis**      | ✅      | GitHub renderiza automaticamente     |
